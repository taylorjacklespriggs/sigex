The entry point for the genetic algorithm is genetic.py.
The number of training iterations and size of the initial pool can be
modified at the top of the file.
The argument -nodisplay will prevent the weights from being
displayed.
The entry point for the evaluation stage is evaluate.py.
The network structure and training iterations can be
modified at the top of the file.
The requirements are
python 3.5.1
numpy 1.11.0
opencv 3.1.0-dev
pyalsaaudio 0.8.2
